# 应用程序

目前采取的方案是将软件安装集中在 包管理工具 和 商店

主要原因有俩个:

1. 方便更新
2. 易于迁移

## 包管理工具

经过一些简单的测试, 目前包管理工具更适合下载一些开源软件和一些 Command-line Interface 工具, 商业软件更适合使用商店来进行集中管理, 下面列举自己常用的软件, 以 Scoop 为例
scoop 相关内容移至[此处](./scoop.md)
